拖大 R

R 表示测量噪声方差。R 越大，滤波器越不信传感器，估计曲线更平滑，但真实值变化时跟随会变慢。

拖大 Q

Q 表示过程噪声方差。Q 越大，滤波器越认为模型预测不靠谱，因此更愿意跟着测量值走。

拖大 P₀

P₀ 是初始估计误差。P₀ 越大，刚开始越“不自信”，初期卡尔曼增益更大，收敛更快。

卡尔曼滤波一维估计仿真器

下面把某一个采样点拆开看：先预测，再算增益，最后用测量值修正。拖动“查看第 k 步”可以逐帧理解每一步在做什么。

k=0 / K=0.000 / P=0.000
1. 预测
x̂k− = x̂k−1
Pk− = Pk−1 + Q
这个一维模型默认“下一刻大概率接近上一刻”，Q 用来承认模型并不完美。
2. 计算增益
Kk = Pk− / (Pk− + R)
K 是权重。K 大，测量值 z 的影响大；K 小，预测值 x̂⁻ 的影响大。
3. 修正估计
x̂k = x̂k− + Kk(zk − x̂k−)
z − x̂⁻ 是“测量残差”。滤波器只吸收其中一部分，吸收比例就是 K。
4. 更新不确定性
Pk = (1 − Kk)Pk−
完成一次测量修正后，滤波器通常会比预测时更自信，所以 P 会下降。
